int CIniFile::GetValueI ( string const  keyname,
string const  valuename,
int const  defValue = 0 
) const

Liest einen speziellen Wert aus der CIniFile aus.

Der Wert der ausgelesen wird, muss durch die Parameter keyname und valuename defferenziert werden. Durch "keyname" wird hier eine grobe Unterteilung in Rubriken gemacht. Die Spezifizierung in einer Rubrik wird dann durch "valuename" vorgenommen.

Parameter
keynameName der Rubrik, aus welcher der Wert ausgelesen werden soll
valuenameName der Variable, die innerhalb einer Rubrik ausgelesen werden soll
defValueStandard Wert, falls kein Wert ausgelesen wurde
Rückgabe
