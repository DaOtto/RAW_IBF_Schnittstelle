char Def_IBF_Allgemein_IST::InitAchse

Flag zur Steuerung des Anfahrens an die Startposition.

Mögliche Werte:

  • 'v' : Interface Test.
  • 'w' : Analyse Reglerverhalten Axialgerüst DA Jenkouk.
  • 'x' : Axialgerüst auf Position 700mm + AG_Regelung::Sprunghoehe anfahren.
  • 'y' : Axialgerüst auf Position 700mm anfahren.
  • 'a' : Positionieren des Radialschlittens und Positionieren des Axialschlittens über Ringhöhe um ein Aufsetzen zu vermeiden.
  • 'b' : Positionieren des Axialgerüsts an Startposition.
  • 'c' : Position Axialgerüst: Profilierungsposition.
  • 'd' : Position Axialschlitten.
  • 'z' : 2 Umdrehungen abwarten für gleichmäßiges Profil über Umfang.

Definiert in Zeile 187 der Datei IBF_struct.hpp.

Wird benutzt von RPW::IstIBF_SollIBF(), LOGGER_RWTH::Kanalliste(), RPW::Main() und TEST_IBF::WRITE_VALUES().
