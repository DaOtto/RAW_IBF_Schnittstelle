int RWTH_EXCH::Main_Control ( void  )

Diese Methode kann in zwei Teile gegliedert werden.

Der erste Teil wird nur ausgefuehrt, wenn das Programm RWTH_Task das erste mal durchlaufen wird. Der zweite Teil wird nur dann durchlaufen, wenn der erste Teil bereits durchlaufen wurde.

Im ersten Teil werden Vorbereitungen für das Walzen getroffen. Hier wird festgelegt, welcher Pfad in der RPW::Main() für die Walzvorbereitung verwendet wird. Dies wird mit Hilfe der Variable Def_IBF_Allgemein_SOLL.InitAchse festgelegt.
